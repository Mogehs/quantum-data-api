Real-time Stock Quotes

Access real-time, low-latency market data for thousands of global equities. Our REST API delivers the latest price, volume, and depth data updated every millisecond.

Example Request

POST /v1/market/quotes
1234567891011
import requests url = "https://api.massive.com/v1/market/quotes" payload = { "symbols": ["AAPL", "MSFT", "NVDA"], "fields": ["price", "change", "volume"] } headers = { "Authorization": "Bearer YOUR_API_KEY" } response = requests.post(url, json=payload, headers=headers) print(response.json())

Available Parameters

symbols Required Array of strings

A comma-separated list of ticker symbols (e.g., AAPL, TSLA).

interval Optional String

Streaming interval for real-time updates. Default is 1s.
