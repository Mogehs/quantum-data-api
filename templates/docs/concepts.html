Understand the fundamental building blocks of the QuantDataApi ecosystem. Mastering these concepts will help you integrate our data more effectively.

label Ticker Symbols

Everything starts with a symbol. A ticker symbol is a unique series of letters representing a particular security listed on an exchange.

US Equities

"AAPL", "MSFT", "TSLA"

Crypto

"BTC", "ETH", "SOL"

function API Functions

The function parameter is the most important part of any request. it tells our engine exactly what type of intelligence you need.

Historical OHLCV data at a daily granularity.

Quote Endpoint

Real-time lightweight price and volume data.

Technical Indicators

Computed values like SMA, RSI, or MACD.
