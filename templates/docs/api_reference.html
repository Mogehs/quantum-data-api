GET /query

TIME_SERIES_DAILY

This API returns raw (as-traded) daily time series (date, daily open, daily high, daily low, daily close, daily volume) of the global equity specified, covering 20+ years of historical data.

info

If you are also interested in split/dividend-adjusted data, please use the Daily Adjusted API.

Query Parameters

Parameter Type Description
function string Required. Set to TIME_SERIES_DAILY
symbol string Required. The ticker symbol of the equity (e.g., IBM)
outputsize string Optional. compact (default) or full.
