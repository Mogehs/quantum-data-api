GET /query

TIME_SERIES_MONTHLY

This API returns monthly time series (last trading day of each month, open, high, low, close, volume) for the equity specified, covering 20+ years of historical data.

Query Parameters

Parameter Type Description
function string Required. Set to TIME_SERIES_MONTHLY
symbol string Required. The ticker symbol (e.g., IBM)
apikey string Required. Your API Key.
