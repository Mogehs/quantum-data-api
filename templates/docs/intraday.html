GET /query

TIME_SERIES_INTRADAY

This API returns intraday time series (date, open, high, low, close, volume) of the equity specified, covering the most recent 1-2 months of intraday data.

info

For intraday data older than 2 months, please refer to our Historical Intraday Archive.

Query Parameters

Parameter Type Description
function string Required. Set to TIME_SERIES_INTRADAY
symbol string Required. The ticker symbol (e.g., IBM)
interval string Required. 1min, 5min, 15min, 30min, 60min.
