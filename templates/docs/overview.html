Rate Limits

To ensure platform stability, we enforce rate limits based on your subscription tier. Exceeding these limits will result in a 429 Too Many Requests response.

Tier Requests / Min Monthly Volume
Free Sandbox 5 requests 1,000 requests
Professional 120 requests 100,000 requests
Enterprise Unlimited Custom

Response Formats

By default, all responses are returned in JSON format. We use standard HTTP response codes to indicate the success or failure of an API request.

info Content-Type

Ensure your requests include the Accept: application/json header to receive structured data.
