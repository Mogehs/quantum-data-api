Quickstart Guide

Get up and running with QuantDataApi in under 5 minutes. This guide will walk you through making your first API call.

1 Get your API Key

If you haven't already, sign up for a free account on our Developer Dashboard and grab your API key from the dashboard overview.

2 Make your first request

Open your terminal and run the following cURL command. Replace DEMO_KEY with your real API key.

Terminal / cURL content_copy 
curl "https://api.quantdata.io/query?function=TIME_SERIES_DAILY&symbol=IBM&apikey=DEMO_KEY"

You should receive a JSON response containing daily price data for IBM.
